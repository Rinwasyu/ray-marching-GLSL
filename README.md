# ray-marching-GLSL

Ray marching program written in C and GLSL.

## Requirements

- GCC
- OpenGL 4.6
- GLEW
- GLFW

## Usage

Run below:
```console
$ git clone https://github.com/Rinwasyu/ray-marching-GLSL.git
$ cd ray-marching-GLSL/src
$ make
$ ./main
```
